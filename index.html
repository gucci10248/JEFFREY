What BioMCP Gives Your AI

🧬 Gene & Variant Lookup

Instant access to ClinVar, gnomAD, MyGene, and OncoKB. Annotate any human gene or variant with clinical significance, population frequency, and functional impact.

📚 Literature Search

Full PubMed, Europe PMC, PubTator3, and LitSense2. Find papers, get structured abstracts, semantic recommendations — all in one call.

🔬 Clinical Trials

Search 450,000+ trials on ClinicalTrials.gov with geo-filters, eligibility criteria, and phase detection. Perfect for systematic reviews.

💊 Drug Intelligence

FDA labels, FAERS adverse events, DDInter interactions, PharmGKB pharmacogenomics, and EMA/WHO regulatory data — all drug truth in one place.

🗺️ Pathway & Enrichment

KEGG, Reactome, and Enrichr g:Profiler. Map genes to pathways and run gene set enrichment analysis directly.

🩺 Disease-Phenotype

Monarch Initiative, HPO, GWAS Catalog. Connect diseases to phenotypes, genes to traits, and run cross-entity discovery.

1-Minute Setup

# Option A: Pull from Docker Hub (recommended)
docker pull jackgucci/biomcp:latest
docker compose up -d

# Option B: Build from source
git clone https://github.com/gucci10248/JEFFREY.git && cd JEFFREY
docker build -t biomcp . && docker compose up -d

# Your MCP server is live
curl http://localhost:8090/health

# Connect Claude Desktop, Hermes, or any MCP client to:
# http://localhost:8090/mcp

Enterprise-Ready Security

🔒 Read-Only

Zero mutation of external data. BioMCP never writes to any upstream biomedical database.

🐳 Non-Root Container

Runs as unprivileged user. Read-only root filesystem. No CAP_SYS_ADMIN.

📦 Minimal Surface

Single static binary. Zero runtime dependencies. No npm, no pip, no gem.

🚫 No Telemetry

Zero outbound calls except to the public biomedical APIs you explicitly query.
